递归求解二叉树高度小记

发表于 2020-02-10 更新于 2020-02-13 分类于 数据结构

程序要求：

要求写一个函数求解二叉树的高度

程序代码：

#include postordergetheight(BinTree BT)
{
    int HL,HR,MAXH;
    if(BT)
    {
        HL = postordergetheight(BT->left);
        HR = postordergetheight(BT->right);
        MAXH = (HL>HR) ? HL:HR;
        return MAXH+1;//返回左右子树中深度最大的个数，再加上父节点，因为不会统计父节点。
    }
    else return 0;
}

个人理解方法：

postordergetheight(BinTree BT)这个函数的作用是给其一个父节点作为参数他会求出由此父节点遍历到结束的深度，这样子的话
HL = postordergetheight(BT->left);
HR = postordergetheight(BT->right);

这两个都只求了有此根节点的左右两个节点为根节点时的深度，最后需要加上1（即加上父节点）。
在遇到递归时，我总是将其看这个递归函数看作一个整体，并且要知道这个函数的作用是什么，并且找到递归边界，也就是试想给这个函数传入什么参数时，应该返回一定的值，这个时候不需要思考要递归多少次，以及怎么递归，只要想第一次就传这个参数会不会到边界。
