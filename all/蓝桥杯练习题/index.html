蓝桥杯练习题

问题描述

Fibonacci数列的递推公式为：Fn=Fn-1+Fn-2，其中F1=F2=1。
当n比较大时，Fn也非常大，现在我们想知道，Fn除以10007的余数是多少。

正确代码：

#include<cstdio>
int main()
{
    long long n;
    long long temp1;
    long long temp2;
    long long  i;
    long long  ans;
    long long  fin_ans;        
    scanf("%ld",&n);
    for(i=1;i<=n;i++)
    {
        if(i==1)
        {
            temp1 = 1;
              ans = 1;
        }
        else if(i==2)
        {
            temp2 = 1;
              ans = 1;
        }
        else
        {
            ans = (temp1+temp2)%10007;
        }
        temp1 = temp2;
        temp2 = ans;
    }    
    printf("%ld",ans);
    return 0 ;
}

编码所遇到的问题：

1、用递归函数求出解，再取余，这样子做的时间复杂度太大。

2、用普通的for循环来编写，求出解再取余，这么做会数字太大越界。

编码的正确思路：

用for循环求解，但是不求出每个数列具体的值，而是求每个具体的值与10007的取余后的数，这么做也可求出解，同时大大减小了结果的值（限制在10007之内）。
